# Security

PillowLock is a practical personal project, not a certified security product.

## Reporting Security Issues

If you believe you found a security issue, please avoid posting full exploit details publicly right away.

Preferred approach:

- use GitHub security reporting if it is enabled for this repository
- if private reporting is not available, open an issue with minimal detail and ask for a private follow-up channel

## Scope Notes

Useful reports include:

- encryption or authentication bypasses
- incorrect file handling that could expose plaintext
- key file or password handling issues
- update or installer trust problems

General bugs and feature requests should go to the normal issue tracker.
