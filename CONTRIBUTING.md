# Contributing

Thanks for taking a look at PillowLock.

This is a personal project and also my first Rust project, so focused, practical contributions are especially helpful.

## Bug Reports

If you open an issue, please include:

- what you tried to do
- what happened instead
- your OS version
- screenshots or error messages if available

## Feature Requests

Feature requests are welcome.

If possible, explain:

- the problem you are trying to solve
- how you expect the feature to work
- whether it affects encryption, restore flow, UI, or packaging

## Pull Requests

Please try to keep pull requests small and focused.

Helpful things to include:

- a short summary of the change
- screenshots for UI changes
- notes about behavior changes
- test results when available

## Local Checks

```bash
cargo build
cargo test
```
